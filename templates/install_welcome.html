First Time Setup

It seems this is the first time you have launched this web application.
In order to start using the application, it needs to be configured first.
This setup wizard will ask you for few details and you are ready to go.


What is required to complete this setup?

  • A MySQL server that you can connect to
  • MySQL User/password with privilege to create database and tables
  • You will be asked to set up a password for MyBanker admin account
  • Optionally you can set up your first user account if you like
